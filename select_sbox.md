# sbox選取依據

根據桐人對青眼惡魔使用星爆氣流斬雙手動作決定(16連擊對應到16個round)。
(因往下的動作較少，將前刺的動作於偶數次出現時對應至 sbox 1)

| 動作         | 對應sbox編號 |
| ------------ | ------------ |
| 前(odd)或上  | 0            |
| 後(even)或下 | 1            |
| 往左         | 2            |
| 往右         | 3            |
| 左上         | 4            |
| 右上         | 5            |
| 左下         | 6            |
| 右下         | 7            |
